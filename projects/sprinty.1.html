Sprinty

Sprinty is an online tool that helps students run design sprints.

Designed around the University of South Wales Graphic Communication course, Sprinty helps users with planning tasks and managing time during design sprints.

THE USER

Graphic Communication students running design sprints

WHAT DOES THE USER NEED?

  • A tool that helps them know what tasks to do and in what order
  • A tool to help them manage how long they spend on each task
  • A tool that is simple, flexible, and is going to be used in a hurry

Sprinty guides the user through each stage of the sprint process.

Each stage is timed proportionally to how long the user has to run the sprint.

Sprinty is flexible and scales to any display. The interface is designed to be used in a rush.
